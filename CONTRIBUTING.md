# Contributing to Vibe Games Extension

Thank you for your interest in contributing! This project is designed for high-speed agentic prototyping.

## How to Contribute

1.  **Issue First:** If you find a bug or have a feature idea, please open an issue first to discuss it.
2.  **Pull Requests:**
    - Fork the repository.
    - Create a feature branch.
    - Ensure your changes follow the existing style and mandates (see `vibe-games.md`).
    - Include license headers in all new source files.
    - Submit a PR with a clear description of the change.

## Contributor License Agreement

By contributing to this project, you agree that your contributions will be licensed under the Apache License, Version 2.0.

## Coding Standards

- **Surgical Edits:** Use the `replace` tool for precise changes.
- **Validation:** Always include tests or validation steps for new features.
- **Documentation:** Update `README.md` and relevant `SKILL.md` files if you change functionality.
