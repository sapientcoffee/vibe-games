# Security Policy

## Supported Versions

We support the latest version of this extension.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it via the project's issue tracker on GitHub. 

**Note:** Do not share sensitive information (like API keys or credentials) in public issues. 

We will investigate all reports and provide a fix as quickly as possible.
