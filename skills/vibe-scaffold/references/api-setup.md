# API Setup Reference

This reference provides the workflow for bootstrapping a FastAPI bridge layer.

## Workflow (for Generalist)
1. **Directory**: Create `<API_DIR>`.
2. **Install**: `uv venv && .venv/bin/pip install fastapi uvicorn`.
3. **Write**: Create a `main.py` with basic health checks and placeholder endpoints.
4. **Launch**: Start uvicorn on `<PORT>`.
