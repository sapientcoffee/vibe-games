# Frontend Setup Reference

This reference provides the workflow for bootstrapping a React + Tailwind frontend.

## Workflow (for Frontend-Expert)
1. **Initialize**: Setup a React + Tailwind CSS project in the specified `<FRONTEND_DIR>`.
2. **Placeholder**: Create a high-fidelity "Coming Soon" or "Vibe Dashboard" in `src/App.tsx`.
3. **Launch**: Start the Vite/React server in the background on `<PORT>`.

## Aesthetic Guidelines
- Clean typography (Inter/Roboto).
- Centered, minimalist layouts.
- Interactive hover states.
- Dark mode preference unless specified.
